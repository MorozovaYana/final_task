Flask + PostgreSQL Airline Demo

Блог + Авіаційна база даних (Flights, Bookings, Logs)
Цей проєкт — це навчальна Flask-аплікація, яка працює з базою даних PostgreSQL.

Вона поєднує:
- просту блог-платформу (entries);
- авіаційний модуль (рейси, бронювання, логування);
- приклад функцій, тригерів і процедур у PostgreSQL;
- HTML-сторінки з мінімальними стилями;
- Flask маршрути для роботи з даними.

  Структрура проекту:

  project/
│
├── main.py               
├── schema.sql            
├── populate.sql          
│
├── templates/
│   ├── index.html
│   ├── flights.html
│   ├── logs.html
│   ├── booking_result.html
│   └── browse.html
│

    └── style.css


  При розробці використала такі технології:
- Python 3
- Flask
- PostgreSQL
- psycopg2
- HTML + CSS
-  PL/pgSQL (тригери, функції, процедури)

Мій застосунок має наступний функціонал:
1. Блог (entries)
   
- Перегляд записів
- Завантаження тестових даних (populate.sql)
- Вивід у вигляді HTML або текстового дампу
  
2. Авіаційний модуль
  
  Таблиці:
- flights — рейси
- bookings — бронювання
- logs — лог дій
  
  Функції, тригери, процедури PostgreSQL:
- log_event(text) — функція для створення запису у logs
- booking_created() — тригер-функція
- trg_booking_created — логування створення бронювання
- make_booking(name, flight_id) — процедура з логікою:
- перевірка кількості вільних місць
- створення бронювання
- зменшення available_seats
- запис у лог
  
  Інтерфейси:
- /flights — список доступних рейсів
- /book — форма бронювання
- /logs — перегляд логів


  Приклад використання

Перейдіть на сторінку /flights, щоб переглянути всі події.
Оберіть рейс та введіть ім’я покупця, а потім натисніть Book,.
Якщо місця закінчились — виведеться помилка.
